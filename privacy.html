Política de Privacidade - SOS Emoções

Versão curta: Este aplicativo armazena dados localmente no dispositivo do usuário para fins de acompanhamento emocional. Dados sensíveis não são enviados a servidores externos por padrão.

Consentimento

Se o usuário for menor de 18 anos, é obrigatório o consentimento do responsável. O consentimento é registrado localmente e pode ser revogado apagando os dados do aplicativo.

Dados coletados

  • Nome, série, turma e turno (informações de perfil)
  • Registros de emoções (data/hora, emoção)
  • Mensagens de chat (armazenadas localmente)
  • Alertas detectados automaticamente (risco)

Contato

Para dúvidas sobre privacidade, contate o mantenedor do projeto.
